RULES FOR ASSIGNMENT SUBMISSION:

  1. Submit only your implementation files. Please do not send the executables,Input or Output files.

  3. Any assignment sent via mail will not be accepted.
  4. The deliverable files must be contained in a folder named with your rollno. e.g 14XX010XX
  5. Send the Folder after compressing it into .tar.gz or .zip archive. e.g 14XX010XX.tar.gz.
  6. Upload the compressed file from here.
  7. Please note that for uploading assignment it is mandatory to use the steps mentioned above as the programs will be checked using scripts. One Sample can be found here.
